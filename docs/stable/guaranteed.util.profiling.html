guaranteed.util.profiling module¶

class guaranteed.util.profiling.Timer(header=None, flush=False, silent=False)¶

Bases: object

Console profiler decorated as Python context manager. Displays the elapsed time of the block after execution.

Parameters

  • header (string) – Displayed header for the executed block.

  • flush (boolean) – When True, the header is displayed before the execution starts. Otherwise, the header is printed after execution with the elapsed time part. Default is False.

  • silent (boolean) – When True, no information is displayed in the console. Default is False.
